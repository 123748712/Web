indexOf(value, start)
  	: value - 찾을 문자열(필수)
  	: start - 검색을 시작할 index지정 (선택/미입력시 처음부터 검색한다)
  	
  	lastIndexOf(value, end)
  	: value - 찾을 문자열(필수)
  	: end - 검색을 끝낼 index지정 (선택/미입력시 끝까지 검색한다)
  	
  	- 대소문자를 구분한다
  	- 찾는 문자열이 없을 경우 -1을 반환한다
