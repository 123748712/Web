Javascript는 웹브라우저가 모든 문서의 요소를 로드한 뒤 실행해야 오류가 발생하지 않는다
  	오류가 발생하는 원인 : 아직 생성되지 않은 html 요소에 접근할 때 
  	해결 
  		window객체의 load 사용
  		- 문서의 모든 콘텐츠(image, script, css, 기타 요소)가 로드된 후 발생하는 이벤트
  		- 문서에 포함된 모든 요소가 로딩된 후 실행되므로 (상대적으로)느리다
  		
  		document객체의 DOMContentLoaded 사용
  		- html과 script가 로딩된 시점에서 발생하는 이벤트
  		- load이벤트보다 먼저 발생(=실행적인 면에서 더 빠름)
