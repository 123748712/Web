변수 키워드 3종 - var, let, const
  	var
  		- scope(유효범위) : 함수범위
  		- 변수 호이스팅이 발생 (=선언 이전 참조 가능한 비정상적 흐름의 코드 작성을 가능케 하는 기능)
  		- 변수 선언시 키워드 생략을 허용 (=의도치 않은 변수의 전역화를 발생시킴)
  		- 재선언과 재할당 가능
  		>> 위의 기능들이 코드의 복잡도를 높이고 오류를 발생하는 등 문제가 다수 발생하여
  		이를 보완하고자 새롭게 만들어진 키워드가 ES6버전의 let과 const이다
  		
  	let
  		- scope : 블록('{}')범위
  		- 변수 사용전까지 초기화 되지 않음
  		- 변수 재선언 불가
  		- 변수 재할당 가능
  		- 변수 호이스팅 불가
  		
  	const
  		- scope : 블록범위
  		- 변수 재선언 및 재할당 불가
  		- 변하지 않는 값을 갖는 변수 생성(=상수 변수)
  		- 선언시 반드시 값이 할당되어야 하며 할당되지 않으면 에러 발생
  		- 강제는 아니지만 보통 대문자를 사용해서 선언
