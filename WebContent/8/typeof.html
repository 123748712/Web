typeof : 변수의 데이터 타입을 반환하는 연산자
  	
  	원시타입(primitive data type)
  	string : 데이터 타입이 문자열
  	number : 데이터 타입이 숫자
  	boolean : 데이터 타입이 불리언
  	null : 지정되지 않은 빈 상태
  	undefined : 변수가 정의되지 않았거나, 값이 없는 상태
  	object(또는 function) : 데이터 타입이 함수 또는 배열 등의 객체인 상태
  	------------------------------------------------------
  	ES6에서 추가된 7번째 원시 타입 Symbol
  	: 유일성이 보장되는 자료형으로 변경 불가능한 원시타입의 값
  	: 이름의 충돌위험이 없는 객체의 유일한 프로퍼티 키를 만들기 위해 사용된다
