객체의 속성과 메소드에 접근하는 방식 2가지
  	1. 객체.key >> 마침표(. dot Notation)를 통해 접근
  	2. 객체["key"] >> 대괄호 안에 key를 문자열 로 넣어 접근
  	* 대괄호 활용시 key를 문자열로 표시하지 않았을 경우(=따옴표 쓰지 않은 경우)
  		변수로 인식하므로 주의해야 한다
  	* 변수로 참조하기
  		let key = "age";
  		myObj[key]
  	
  	객체생성 2가지 방식 중 더 권장되는 리터럴 생성 방식
  	- 코딩이 더 편하고 가독성이 좋음
  	- 속도가 빠름(다수의 객체 생성후 시간 비교시 차이가 난다고 함)
  	- 자바스크립트 오버라이딩 방지(생성자 방식으로 생성시 오버라이딩 가능성이 있음)
